# GREME


## 🎠 About
- 친환경적인 생활을 실천하고 기록할 수 있도록 돕는 챌린지 & 다이어리 플랫폼
- 매일매일의 친환경적인 생활을 다이어리에 기록하고 타인과 정보를 공유하며, 챌린지 기능을 통해 다양한 사람들과 함께 챌린지에 도전하여 친환경적인 생활을 보다 더 즐겁게 실천할 수 있도록 도와주는 서비스입니다.
